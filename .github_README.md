# BlazePlayer Project

## ⚠️ IMPORTANT: READ THIS FIRST!

This project requires Firebase configuration before it will run.

### Quick Start

1. **Read**: `QUICK_START.md` for essential setup steps
2. **Detailed Guide**: `SETUP_GUIDE.md` for complete instructions
3. **Must Have**: `android/app/google-services.json` from Firebase

### Without google-services.json, the app will crash!

- ✅ **Clean Architecture** - Feature-based organization
- ✅ **Firebase Authentication** - Email, Google, Facebook

- ✅ **Play Store Ready** - Release build configuration

## Quick Setup

```bash
# 1. Get dependencies
flutter pub get

# 2. Add google-services.json to android/app/

# 3. Run the app
flutter run
```
